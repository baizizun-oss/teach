# KNN 算法（K-最近邻）

## 1. 核心思想  
> “物以类聚”——待测样本的类别由其**最近的 K 个邻居**投票决定。

## 2. 训练阶段（无显式训练！）  
- 将所有已知类别的样本 **按特征向量化**（如：[身高, 体重] → [170, 65]）  
- **保存全部样本数据**（KNN 是“懒惰学习”，不建模，只存储）

## 3. 识别/预测阶段  
1. 对新样本提取相同特征，得到特征向量  
2. 计算它与**所有训练样本**的**距离**（常用欧氏距离）  
   $$
   d = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}
   $$
3. 找出距离最近的 **K 个样本**  
4. 由这 K 个邻居**多数投票**决定新样本的类别

## 4. 样本点分布示意图（建议手绘或PPT展示）

     ▲
     |        ● A类
     |     ●       ○ B类
     |        ? ← 待分类点
     |   ●      ○
     |        ○     ●
     +------------------→ 特征1（如身高）
           特征2（如体重）

- 图中：● 表示类别 A，○ 表示类别 B  
- “?” 是待分类点  
- 若 K=3，圈出最近的 3 个点（如 2 个 ○ + 1 个 ●）→ 判为 B 类
